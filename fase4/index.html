DISEÑO DE SITIOS WEB

El presente sitio web está elaborado como parte de las actividades del curso de diseño de sitios web, y en el estudiaremos los conceptos relacionados con el desarrollo y diseño de páginas web con las tecnologías más recientes y utilizadas para esto.

Del mismo modo este sitio está desarrollado con HTML5 y siguiendo los conceptos de desarrollo y diseño de sitios web ilustrando así los temas que serán explicados.

Como se ha dicho anteriormente el propósito de este sitio es mostrar los principales temas referentes al desarrollo desde etiquetas básicas en HTML hasta estilos en CSS, apoyándose con imágenes y demás elementos multimedia.

Este sitio busca describir las etiquetas básicas de HTML, incluyendo sus atributos y valores.

También se puede conocer más sobre el autor de este sitio en el respectivo enlace en el menú de navegación que esta sobre este contenido.

Los temas que trataremos en las siguientes páginas de este sitio son:

Etiquetas Básicas HTML

Del lenguaje que define la estructura de los sitios WEB que traduce como Lenguaje de etiquetas de hipertexto, repasando las etiquetas que lo componen su función, aplicaciones y los atributos que usan.

Atributos y valores HTML

Estos son valores que se añaden a las etiquetas HTML y definen su comportamiento y apariencia según los criterios del desarrollador.
