Atributos y valores HTML

Son elementos añadidos a las etiquetas HTML estos se ubican en la etiqueta de apertura, para servir de complemento a la información con la que se personaliza el funcionamiento y apariencia de las etiquetas.

Estos se escriben con el nombre del atributo=”valor” de forma que se pone el nombre del atributo seguido del signo igual (=) y entre comillas el valor, dentro de la etiqueta de apertura.

Por ejemplo:

<p width="400px" height ="200px" >Este es un párrafo</p>

Valores de un atributo

Al usar los atributos estos deben contener un valor, siendo una cantidad característica o un valor que se ajuste a dicho atributo, como se ha mencionado antes esta se escribe entre comillas después del signo igual y estos adquieren varios valores según este disponible.

Entre los atributos y valores más usados tenemos:

  • Id: identifica de manera única un elemento, este puede ser cualquier palabra que sirva para identificar el elemento.
  • Class: identifica multiples elementos, los cuales pertenecen a una misma clase, igual que el id puede ser cualquier palabra.
  • Align: define la alineación del elemento dentro del contenido del sitio, estos son direcciones o ubicaciones como “right”, “center”, “left” y “justify”.
  • Border: permite darle borde a los elementos html, sus valores definen el grosor y estilo de los bordes con “solid”,”dotted”, “dashed”, “doublé”, etc.
  • Style: para definir el estilo del contenido y sus etiquetas.
  • Background-color:define el color de fondo de elementos como la etiqueta body, y sus valores pueden cualquier nombre de color o colores en formato hexadecimal, RGB y cualquier otro compatible como por ejemplo: “red”, “blue”, “ffffff”, etc.
  • Href: rutas y enlaces HTML, que se definen con una ruta URL.
  • Height: la altura del elemento, se definen con unidades de medida en distancia, pixeles o porcentaje, por ejemplo 100%, 12em, 10px.
  • Width: el ancho del elemento, como con Height esta se define en unidad de medida.
  • Src: ruta de las imágenes, audios, videos y otros elementos multimedia, se define con una ruta URL.

Ejemplo

Aplicando el atributo style a una etiqueta img con distintos estilos de borde, con un ancho y alto fijos.

<img src="" width=200px height=200px style="background-color=blue>
